



# EditKaro.in Portfolio Website

A modern, responsive portfolio website for EditKaro.in showcasing video editing services, portfolio projects, testimonials, and company information.

## 🚀 Features

- Modern dark-themed UI
- Fully responsive design
- Interactive portfolio filtering
- Video lightbox previews
- Smooth scrolling navigation
- Animated sections and transitions
- Mobile-friendly layout
- Service showcase section
- Client testimonials
- Call-to-action section

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts
- YouTube Embed API

## 📂 Project Structure

```
EditKaro/
│
└── index.html
```

## ▶️ Running the Project

### Option 1: Open Directly

Simply open:

```bash
index.html
```

in any modern browser.

### Option 2: Run Using Python Server

```bash
py -m http.server 8000
```

Then visit:

```
http://localhost:8000
```
